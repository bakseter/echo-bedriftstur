## Development setup

**1. Clone the git repository**
    
    git clone https://github.com/bakseter/echo-bedriftstur

**2. Install dependencies**

    cd echo-bedriftstur && yarn

**3. Serve the website locally with**
    
    yarn dev

**4. ... or build it with**

    yarn build
